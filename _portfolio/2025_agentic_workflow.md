**2025 – present**

Multi-agent orchestration framework powering both industry work and PhD research. Coordinates parallel workstreams across multiple AI backends, manages task dependencies, and automates deployment and reporting.

- Multi-backend coordination with automatic failover
- Parallel sub-agents that run independently and report on completion
- Task dependency management with automatic downstream launch
- Persistent memory and context recovery across sessions
- Bidirectional messaging bridge for status and coordination

The system maintains a structured knowledge base that grows across sessions. New agents inherit all prior context.
